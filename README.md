
# Overview

A Python script to rapidly generate new data points for the JUNE/JULY preset ai copilot.

<br>

# Installation Guide
Open a terminal session and run the following commands. You will be prompted to paste in the path to your inference.atp file, which should have been made by the JUNE/JULY preset ai copilot.
```
git clone https://github.com/tgarvs/RapidFireJunePresets.git
cd RapidFireJunePresets
make setup
```

<br>

# How to Use

1. cd to proper directory
2. Run the following command in terminal
```
make run
```
3. Open JUNE so it is easily accessible
4. Follow instructions in terminal
